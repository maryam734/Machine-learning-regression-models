# Machine Learning Regression Models

## Overview
This project demonstrates the implementation of Linear Regression and Logistic Regression using Python.

## Objectives
- Understand supervised machine learning techniques.
- Implement regression and classification models.
- Analyze and interpret model outputs.

## Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

## Models Implemented
1. Linear Regression
2. Logistic Regression

## Results
The project successfully trains and evaluates Linear and Logistic Regression models. Relevant output screenshots are included.
